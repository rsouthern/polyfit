# polyfit
Simple GTest example using TDD on fitting a polynomial to data.

## Getting Started
### Requirements
* GTest (https://github.com/google/googletest) - should download if not found.
* Eigen

### Installation
```````
mkdir build
cd build
cmake ..
make -j
```````

## Options
No options!
